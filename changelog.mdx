# **v1.0.0** - 30 de Novembro de 2025

Primeira versão estável da Social API.

**Autor:** Pedro Del Valle

## Funcionalidades

- Conexão de contas Instagram e TikTok
- Sincronização automática de perfis e posts
- Dashboard consolidado com métricas
- Analytics por post com histórico
- Sistema de jobs assíncronos
- Download de vídeos TikTok
- Métricas enriquecidas com IA (Gemini)

## Segurança

- Autenticação via `X-Internal-Secret`
- Multi-tenant via `X-Enterprise-Id`
- Rate limiting por tenant

## Infraestrutura

- Google Cloud Run
- PostgreSQL via Supabase
- Apify para coleta
- Google Gemini para IA
